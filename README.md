# Image Processing Toolkit (C++ + OpenCV)

A mini computer vision preprocessing toolkit implemented in C++ using OpenCV.

## Features
- Grayscale conversion
- Gaussian blur (noise reduction)
- Sobel edge detection
- Canny edge detection
- Grid visualization of pipeline results
- Save processed output images

## Pipeline
1. Load image
2. Convert to grayscale
3. Apply Gaussian blur
4. Perform Sobel edge detection
5. Perform Canny edge detection
6. Display all results in grid view

## Tech Stack
- C++
- OpenCV
- Visual Studio

## How to Run
1. Place `test.png` in executable directory
2. Build in x64 Debug/Release
3. Run program to see grid visualization
